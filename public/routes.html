Editing Routes

Click on the map to add points. Use Undo to remove the last point, Redo to re-add it, and Clear to reset the path.

Select an existing route to modify it or enter a new ID and name to create one. Change the Name field to rename a route. Use Remove to delete the selected route.

After drawing a rough path, click Snap to Roads to align it smoothly to streets using the Google Roads API.

Press Save to store your changes or Cancel to revert to the last loaded route.
